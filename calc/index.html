TI-30XS MultiView Guide

Quick Start

This is a high-fidelity simulator of the TI-30XS MultiView™. It supports multi-line calculations, fractions, statistical data, and table functions.

Try These Expressions

  • Fraction Addition: Press 1 $\rightarrow$ n/d $\rightarrow$ 2 $\rightarrow$ ▶ $\rightarrow$ + $\rightarrow$ 1 $\rightarrow$ n/d $\rightarrow$ 3 $\rightarrow$ enter. The result is 5/6. Press [◄►] to toggle to decimal format.
  • Exact Radicals: Press 2nd $\rightarrow$ x² (which triggers √) $\rightarrow$ 8 $\rightarrow$ enter. The result is 2√2. Toggle to view 2.8284...
  • Function Table: Press table. Enter a function like 3x² - 5 (use x,y,z,t key for x, and ^ for power). Set Start=0, Step=1. Click Auto $\rightarrow$ OK to view values!
  • Data Editor: Press data. Enter 10, 20, 30 in L1. Press 2nd $\rightarrow$ data (STAT). Select 1-Var Stats, choose L1, click CALC. Scroll to see mean ($\bar{x}=20$), sum, stddev, and min/max!
